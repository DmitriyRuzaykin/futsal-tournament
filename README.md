# Кубок Elita по футзалу ⚽

Веб-приложение для отслеживания результатов турнира по футзалу в Волжском муниципальном районе.

## Возможности

- 📊 **Турнирная таблица** - отображение текущего положения команд в группах A и B
- ⚔️ **Расписание матчей** - просмотр всех матчей с фильтрацией по турам, группам и статусу
- 🔧 **Администрирование** - защищенный вход для ввода результатов и сброса матчей
